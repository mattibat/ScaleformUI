# HUD and Notifications API

ScaleformUI provides a static API to manage in-game HUD elements, notifications, markers, and 3D text drawing. This reference index lists all available HUD features and their respective documentation pages.

## Notification Types

- **[Standard Notification](./ShowNotifications.md)**
  Displays standard feed notifications with customisable text, colours, and briefing options.
  
- **[Advanced Notification](./AdvancedNotification.md)**
  Displays detailed notifications with support for custom icons, backgrounds, flashing colours, and sound effects.
  
- **[Help Notification](./HelpNotification.md)**
  Displays standard help text prompts in the top-left corner of the screen, supporting both per-frame rendering and timed displays.
  
- **[Floating Help Notification](./FloatingHelpNotification.md)**
  Displays help text prompts floating in 3D space at specific coordinates.
  
- **[Stats Notification](./StatsNotification.md)**
  Displays progress bar updates for player stats, showing the transition from old values to new values.
  
- **[VS Notification](./VSNotification.md)**
  Displays versus comparison banners between two players with customisable background colours.

## World Elements

- **[Markers](./Markers.md)**
  Handles 3D world markers with customisable types, positions, scales, colours, and behaviours such as bobbing or rotating.
  
- **[DrawText3D](./DrawText3D.md)**
  Draws text directly in the 3D game world at specified coordinates with customisable fonts, sizes, and colours.
