# Scaleforms API

ScaleformUI provides a static user-friendly API to manage different scaleforms within your game. This reference index lists all available scaleforms and their respective documentation pages.

## Available Scaleforms

- **[Big Message](./BigMessage.md)**
  Displays large, customisable messages on the screen, such as mission passed banners, rank up notifications, and weapon purchase alerts.
  
- **[Countdown](./Countdown.md)**
  Handles in-game countdown sequences with customisable starting numbers, colours, and audio cues.
  
- **[Feed](./Feed.md)**
  Manages the customisable feed system for displaying notifications and updates.
  
- **[Instructional Buttons](./Instructionalbuttons.md)**
  Draws customisable instructional button prompts at the bottom of the screen to guide player inputs.
  
- **[Midsize Message](./MidsizeMessage.md)**
  Displays medium-sized message banners, including coloured shards with custom text and backgrounds.
  
- **[Minimap Overlays](./MinimapOverlays.md)**
  Manages custom overlays and elements drawn directly on the minimap.
  
- **[Mission Selector](./MissionSelector.md)**
  Provides a customisable interface for selecting missions or activities.
  
- **[Player List](./PlayerList.md)**
  Displays a list of players in the session with customisable details and layouts.
  
- **[Rank Bar](./rankbar.md)**
  Displays the player's rank progress bar with customisable experience limits, values, and animation speeds.
  
- **[Warning Screen](./warningscreen.md)**
  Displays a customisable warning screen with support for instructional buttons to handle player decisions.

## New and Upcoming Scaleforms

- **[Multiplayer Chat](./MultiplayerChat.md)**
  Handles custom multiplayer chat interfaces and message rendering.
  
- **[Scaleform Widescreen](./ScaleformWideScreen.md)**
  Manages widescreen-specific scaleform layouts and adjustments.
  
- **[Pause Menu](../PauseMenu/MainView.md)**
  Provides a customisable pause menu interface with support for tabs, settings, and lists.
